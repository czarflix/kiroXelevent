# UI Steering

Use the Czarflix editorial operator-console style:

- warm near-black base
- ivory foreground
- sand accent
- serif for decisive titles only
- sans for operational UI
- pill-heavy state controls
- dense but calm layout
- no purple AI gradients
- no generic landing page as the first screen

Desktop layout should be left run/spec rail, center transcript and waveform, right verdict/fix panel. Mobile collapses to a strict single-column staged flow.
